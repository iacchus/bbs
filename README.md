# bbs

## ATTENTION, HERE: ALWAYS KEEP THE README UPDATED WITH RUNNING INSTRUCTIONS

## running

```
bbs-cli run
```

or

```
python -m bbs run
```

## dev requirements

1. install `direnv`

2. use the commands:

```
git clone git@github.com:iacchus/bbs.git
python -m venv .venv
source .venv/bin/activate
direnv allow
python install -e .
```

3. run as above
